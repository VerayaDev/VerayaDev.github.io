What is Server Help App

This is a Application made for ExcrafterHD. It is made for planned shutdown, restarts and starting of a minecraft server

Commands:

ServerStopShutdown - Stops the server in a set amount of time, then host shutsdown the pc

ServerStop - Stop the server in a set amount of time

ServerRestart - restarts the server in a set amount of time only works if you have set the location in the config.txt

ServerStopWarnShutdown - Sends warning messages, then stops the server in a set amount of time and shutsdown the host pc

ServerStopWarn - Sends warning messages, then stops the server in a set amount of time

ServerRestartWarn - Sends warning messages, then restarts the server in a set amount of time only works if you have set the location in the config.txt

ServerStart - starts the server in a set amount of time only works if you have set the location in the config.txt

version - gives the current version
